# PLC alapú autómosó vezérlés – TwinCAT projekt

Ez a projekt egy egyszerű autómosó rendszer PLC-s vezérlésének megvalósítása TwinCAT környezetben.  
A feladat egyetemi házi feladatként készült, a cél egy több lépésből álló automatizált folyamat modellezése és vezérlése volt.

## A projekt során főleg az alábbiakkal foglalkoztam:

- PLC programozás TwinCAT környezetben
- Ladder Diagram (LD) alapú vezérlés
- funkcióblokkok létrehozása
- állapot alapú folyamatvezérlés
- egyszerű automatizálási logika felépítése

A vezérlés több különálló funkcióblokkra lett bontva, hogy a rendszer átláthatóbb és könnyebben bővíthető legyen.
